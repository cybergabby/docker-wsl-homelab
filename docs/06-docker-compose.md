# Docker Compose

## Overview

Docker Compose allows multiple containers to be managed using a single YAML configuration.

Instead of running several docker run commands manually, Docker Compose automates the deployment.

---

# Start Services

```bash
docker compose up
```

---

# Build Images

```bash
docker compose up --build
```

---

# Stop Services

```bash
docker compose down
```

---

# View Running Containers

```bash
docker compose ps
```
